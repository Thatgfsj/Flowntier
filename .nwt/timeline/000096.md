# 000096 — 自定义中转站实际是 dead code (4 个字段 mismatch) + resolve_role 缺 fallback

## 主席反馈
1. "我的 api 是没变过的" — 401 不是 chairman 的 key 出错
2. "已经保存的 api 没办法修改" — 保存到 role_overrides 的
   mimo:mimo-2.5-pro 路径 chair 无法改
3. UI 显示 "添加自定义中转站" 表单 (ID=mimimu, BaseURL=
   https://token-plan.cn.xiaomimimo.com/v1, kind=openai-sdk,
   models=mimo-v2.5-pro/mimo-v2.5-pn/100000i+)

chairman 想用 token-plan.cn 的中转服务, 但 Flowntier:
- 没把它保存到 custom_provider 表
- resolve_role 不会查 custom_provider 表

## 静态分析发现 2 个 bug

### Bug A: Tauri shell 传 body 字段 vs handler 读的字段 完全不一致

Tauri `add_custom_provider` 发送:
```json
{ "id": "mimimu", "display_name": "...", "kind": "...",
  "base_url": "...", "api_key_env": "CUSTOM_MIMIMU_API_KEY",
  "models": [...] }
```

Pipe server `add_custom_provider` 期望:
```json
{ "name": "...", "base_url": "...", "kind": "...",
  "default_model": "...",
  "api_key": "<RAW value, NOT env var name>" }
```

**4 个字段全部 mismatch**:
- `id` (shell) vs `name` (handler) — handler 返回 400 missing
- `display_name` vs `name` — handler 用第一个 `name` 字段
- `api_key_env` vs `api_key` — env var name 被丢失, raw key 不传
- `models[]` 完全被忽略

**结果**: 自定义中转站的 POST 永远 400 失败或返回 201 但表是空的。
验证: `python` 查 `custom_provider` table → 0 rows.

### Bug B: resolve_role 只查 PRESETS, 不查 custom_provider

```rust
let preset = match crate::providers::get(&provider_short) {
    Some(p) => p.clone(),
    None => return Err(format!("unknown provider preset '{}' ...")),
};
```

如果 chairman 设 `mimimu:mimo-v2.5-pro`:
- `provider_short = "mimimu"` 
- `crate::providers::get("mimimu")` 返回 None (只有 9 个内置)
- 直接报错 "unknown provider preset 'mimimu'"

即使 Bug A 修了, chairman 还是调不通。

## 修复

### Fix A: 修 handler 字段映射

`crates/pipe-server/src/handlers.rs:add_custom_provider`:
- 读 `id`, `display_name`, `api_key_env`, `models[]`
- 持久化 models via `repo.put_model_cache("custom:<id>", models_json)`
- `default_model = models[0].id` (or explicit)
- Secret 保存用 env var name (不是 raw value)

### Fix B: resolve_role fallback to custom_provider

`crates/pipe-server/src/handlers.rs:resolve_role`:
```rust
let (base_url, api_kind, secret_name) =
    if let Some(preset) = crate::providers::get(&provider_short) {
        (preset.base_url, preset.kind, preset.secret_name)
    } else {
        // v0.4.22 (event 000096): look up the custom
        // provider row by id.
        let cp = state.repo.get_custom_provider(&provider_short).await?
            .ok_or_else(...)?;
        let secret = format!("CUSTOM_{}_API_KEY", provider_short.to_uppercase());
        (cp.base_url, cp.kind, secret)
    };
```
