"交付摘要 provider error: provider returned 401: ..."

- orchestrator 的每个 agent 都用 fail message 立刻 结束

- chairman 看不到真正的 401 错误

2. `applyEvent`: Done event status.starts_with('FAILED:') → capture summary  
3. 顶部 dashboard 渲染红色 `Card` 显示 workflowError + 重设提示
4. polling 失败时优先用 workflowError 替代 "工作流超时"
