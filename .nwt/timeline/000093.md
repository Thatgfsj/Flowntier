# 000093 — Revert NSIS taskkill v2 (system freeze)

## 背景
event 000092 加了 Sleep 3000 + retry taskkill. 主席反馈:
"有东西溢出导致系统卡死"。

## 根因分析
NSIS `Function .onInit` 是 single-threaded; 
`Sleep 3000` 会把整个 installer UI thread block 3 秒。
加上 二次 taskkill (1秒 + 第二次 taskkill) = 4-5 秒无响应。
主席看不到 NSIS 进度多次双击安装包 → **进程风暴**:
多个 NSIS instance 同时跑 taskkill → 影响 system 文件锁 → OS lag.

## 修复
**回退到 v0.4.21 单 pass**:
- 一次 taskkill
- 没有 Sleep
- 没有 retry

文件锁错误可能在慢盘上复发, 但那比"系统卡死"好得多
—— 主席可以手动关 Flowntier 再重试。
